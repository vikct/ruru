# Ditch SQLite for Neon PostgreSQL Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Transition the Ruru POS database persistent store from a local SQLite file to a hosted Neon PostgreSQL instance, with connection resiliency and automatic migration execution on application startup.

**Architecture:** Replace SQLite references with `Npgsql.EntityFrameworkCore.PostgreSQL` under PostgreSQL connection strings in the backend, configure EF Core connection retries, and delete and recreate EF Core migrations natively for PostgreSQL.

**Tech Stack:** ASP.NET Core 10.0, Entity Framework Core 10.0, Neon PostgreSQL, Carter, MediatR.

## Global Constraints
- Target Framework: `net10.0`
- Database: Neon PostgreSQL (Free Tier)
- EF Core PostgreSQL Provider: `Npgsql.EntityFrameworkCore.PostgreSQL` version `10.0.2`

### Task 1: Configure EF Core Connection Resiliency

**Files:**
- Modify: `backend/src/Infrastructure/DependencyInjection.cs`

**Interfaces:**
- Consumes: Connection string from configuration.
- Produces: Resilient DbContext registered in DI container.

- [ ] **Step 1: Write the Dependency Injection changes**
  Modify [DependencyInjection.cs](file:///Users/victortan/Repos/ruru/backend/src/Infrastructure/DependencyInjection.cs) to use `sqlOptions.EnableRetryOnFailure()` for the Npgsql configuration.

  Replace:
  ```csharp
          services.AddDbContext<ApplicationDbContext>(options =>
          {
              if (connectionString.Contains("Host=") || connectionString.Contains("Server=") || connectionString.Contains("Port="))
              {
                  options.UseNpgsql(connectionString);
              }
              else
              {
                  options.UseSqlite(connectionString);
              }
          });
  ```
  With:
  ```csharp
          services.AddDbContext<ApplicationDbContext>(options =>
          {
              if (connectionString.Contains("Host=") || connectionString.Contains("Server=") || connectionString.Contains("Port="))
              {
                  options.UseNpgsql(connectionString, sqlOptions =>
                  {
                      sqlOptions.EnableRetryOnFailure(
                          maxRetryCount: 5,
                          maxRetryDelay: TimeSpan.FromSeconds(3),
                          errorCodesToAdd: null);
                  });
              }
              else
              {
                  options.UseSqlite(connectionString);
              }
          });
  ```

- [ ] **Step 2: Build the project to verify compilation**
  Run: `dotnet build backend/src/Infrastructure/Ruru.Infrastructure.csproj`
  Expected: Build succeeds with 0 errors.
